# n8n-nodes-bulldoc

This is an n8n community node that lets you to integrate [Bulldoc.dev](https://bulldoc.dev/) API into your workflows.

[Bulldoc.dev](https://bulldoc.dev/) is a developer-focused service for creating and editing printable documents. Generate invoices, agreements or any other documents from dynamic templates, automate filling out PDF forms, merge, split and edit PDF documents.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

This node supports the following operations:

- **PDF**
  - Generate from a template
  - Fill out a form
  - Merge multiple files
- **Tasks**
  - Get one
  - Get many

## Credentials

The node uses **API key** authentication. You need to create an account on [Bulldoc.dev](https://bulldoc.dev/) and generate a key in your account settings.
